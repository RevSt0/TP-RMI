# TP-RMI
## Menus
Featuring 2 menus:
- Gestion des salles :
    - Ajouter salle
    - Modifier salle
    - Supprimer salle
    - Afficher tous les salles
    - Afficher tous les machines d'une salle
- Gestion des machine :
    - Ajouter machine
    - Modifier machine
    - Supprimer machine
    - Afficher tous les machine
